## 👨‍💻 About Me

Bilgisayar Mühendisliği öğrencisi olarak, karmaşık sistemlerin otomatize edilmesi ve ölçeklenebilir altyapıların modern standartlarda kurgulanması üzerine odaklanıyorum.

* **DevOps Focus:** AWS, Docker ve Kubernetes ekosisteminde altyapı yönetimi, CI/CD süreçleri ve GitOps prensipleri üzerine çalışmalar yürütüyorum.
* **Development:** ASP.NET Core ile performanslı backend sistemleri tasarlıyor ve Swift/SwiftUI ile modern mobil uygulama standartlarını uyguluyorum.
* **Goal:** Üretim standartlarında (production-grade), sürdürülebilir ve yüksek erişilebilirlikli mimariler geliştirmek.
